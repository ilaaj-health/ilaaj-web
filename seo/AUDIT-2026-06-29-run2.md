## What changed this run

### ✅ New fix shipped — [PR #54](https://github.com/ilaaj-health/ilaaj-web/pull/54)
**`fix(seo): align sitemap /pk hreflang to en-PK`.** PR #53 relabeled `/pk` pages as
`en-PK` at the page level (`SEO.astro`), but `src/pages/sitemap.xml.ts` still advertised those
same URLs as `ur-PK`. Conflicting page-vs-sitemap hreflang makes Google ignore the annotation,
partially undoing PR #53. Relabeled the `/pk` alternates to `en-PK` in the sitemap (static +
blog clusters); `/ur` keeps `hreflang="ur"`. Build verified — sitemap emits **213 `en-PK` / 0
`ur-PK` / 3 `ur`**, 303 pages.
